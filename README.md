# Hazelnut

Development documentation for the Kotlin backend toolchain. Helps Java and TypeScript developers adapt to Kotlin environments and Spring Boot.

---

## Guides

| Guide | Who it's for |
|-------|-------------|
| [Java → Kotlin](java-to-kotlin.md) | Java developers migrating to Kotlin |
| [TypeScript → Kotlin](typescript-to-kotlin.md) | Frontend/fullstack engineers moving to backend Kotlin |
| [Spring Boot Cheat Sheet](springboot-cheatsheet.md) | Anyone building Spring Boot services in Kotlin |

---

## Java → Kotlin

Covers the full migration surface: variable declarations, null safety, classes, data classes, sealed classes, coroutines, collections, extension functions, and Java interop. Side-by-side Java and Kotlin examples throughout.

**Start here if:** you're a Java developer touching Kotlin for the first time.

---

## TypeScript → Kotlin

Maps TypeScript concepts to Kotlin equivalents — type system, interfaces, generics, union types → sealed classes, async/await → coroutines, and more. Highlights where the two languages diverge.

**Start here if:** you're a frontend or fullstack engineer moving to the Kotlin backend.

---

## Spring Boot Cheat Sheet

Production-ready Spring Boot patterns in Kotlin: project setup, REST controllers, request handling, services, Spring Data JPA, exception handling, validation, configuration, security, testing, and coroutines. Includes annotation and query-method reference tables.

**Start here if:** you're building or maintaining a Spring Boot service.
